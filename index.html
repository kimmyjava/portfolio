Business Problem
& AI Solution

1. Business Problem
E-commerce customer service teams struggle to respond to WhatsApp inquiries quickly and consistently, leading to delayed responses, missed follow-ups, and frustrated customers — especially during peak hours.

2. How I Used AI to Improve It
I built an automated WhatsApp customer service bot that handles common inquiries instantly using a Groq-powered AI agent with conversation memory, while routing complex complaints to a human agent when needed.

3. Tools Used
n8n for workflow automation — Groq API for AI responses — greenAPI for WhatsApp integration — Simple Memory for per-customer conversation history.

4. Workflow
Incoming WhatsApp message → Extract customer message + chatId → AI Agent generates context-aware reply → IF human flag is set → route to human agent → ELSE → send AI reply back via WhatsApp.

5. Risk & Mitigation
The AI may occasionally give incorrect or misleading product information. To handle this, I built a human escalation node — if the AI detects it cannot confidently answer, it flags the conversation and routes it to a human agent automatically.
